# Real Estate Property Management System

## Project Overview

The Real Estate Property Management System is a Salesforce application developed to manage properties, tenants, lease agreements, vendors, and maintenance requests. The application also provides reporting and dashboard capabilities for property management.

---

## Features

### Property Management
- Create new properties
- Upload multiple property images
- Store property details
- Property listing with server-side pagination
- Filter properties by:
  - Rent
  - Availability Status
  - Furnishing Status

### Tenant Management
- Create and manage tenants
- Assign tenants to properties

### Lease Agreement Management
- Create lease agreements
- Track agreement terms
- Track agreed monthly rent
- Track start and end dates
- Automated email reminder sent one month before lease expiry

### Vendor Management
- Manage vendor details

### Maintenance Request Management
- Create maintenance requests
- Automatically assign the vendor with the least workload

### Automation
- Task creation when a property is assigned to a tenant
- Scheduled Apex for lease expiry reminders

### Reports & Dashboard
- Lease agreements expiring in the next 30 days
- Maintenance requests grouped by status
- Property occupancy rate
